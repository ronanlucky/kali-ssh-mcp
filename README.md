## рҹҡҖ Setup

### Prerequisites

```bash
pip install paramiko
```

- Python 3.10+
- [Claude Desktop](https://claude.ai/download) with MCP support
- Kali Linux host reachable over SSH

---

### 1пёҸвғЈ Clone

```bash
git clone https://github.com/ronanlucky/kali-ssh-mcp.git
cd kali-ssh-mcp
```

### 2пёҸвғЈ Configure SSH target

Edit the top of `mcp_bridge.py`:

```python
SSH_HOST = "192.168.x.x"    # Your Kali IP
SSH_PORT = 22
SSH_USER = "kali"
SSH_PASS = "kali"           # вҡ пёҸ Use key-based auth in production
```

### 3пёҸвғЈ Add to Claude Desktop

Open **Claude Desktop вҶ’ Settings вҶ’ Developer вҶ’ Edit Config** and add:

```json
{
  "mcpServers": {
    "kali-ssh": {
      "command": "C:\\Path\\To\\python.exe",
      "args": ["C:\\Path\\To\\mcp_bridge.py"]
    }
  }
}
```

### 4пёҸвғЈ Restart Claude Desktop

The `kali-ssh` server should appear as **рҹҹў running** in Developer settings.
